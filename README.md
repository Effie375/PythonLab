# PythonLab Examples

Small PythonLab programming language projects.

The course includes teaching in:

* general principles of program design
* algorithms
* logic flowcharts
* algorithm design techniques
* algorithmic problem solving

Language syntax:

* variables
* constants
* expressions
* basic data types
* sentences
* operators
* data input / output
* flow control commands
* repetitions
* lists
* subroutines

## 9.2 Ταξινόμηση

---

```python
for i in range(1,5):
    for j in range(4, i-1, -1):
        if list[j-1] > list[j]:
            temp = list[j-1]
            list[j-1] = list[j]
            list[j] = temp
```

## 9.3 Παράδειγμα ταξινόμησης

---

Να γραφεί ένα πρόγραμμα το οποίο θα δέχεται πέντε αριθμούς, θα τους ταξινομεί σε αύξουσα σειρά και θα τους εμφανίζει.

```python
list = []
for i in range(5):
    num = int(input("Dose stoixeia: "))
    list.append(num)
print(list)
for i in range(1,5):
    for j in range(4, i-1, -1):
        if list[j-1] > list[j]:
            temp = list[j-1]
            list[j-1] = list[j]
            list[j] = temp
print(list)
```

Υπερχείληση CSS

Mπορείτε επίσης να αλλάξετε την υπερχείληση περιεχομένου οριζόντια ή κάθετα.

Το overflow-x καθορίζει τι να κάνετε με την αριστερή / δεξιά άκρη του περιεχομένου.

Το overflow-y καθορίζει τι πρέπει να κάνετε με τα πάνω / κάτω άκρα του περιεχομένου.

Μπορείτε να χρησιμοποιήσετε την ιδιότητα υπερχείλησης όταν θέλετε να έχετε καλύτερο έλεγχο της διάταξης. Η ιδιότητα υπερχείλησης καθορίζει τι θα συμβεί έαν το περιεχόμενο υπερχειλίσει ένα πλαίσιο ενός στοιχείου.

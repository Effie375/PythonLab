## 5.1 Μονοδιάστατη λίστα

---

- Δυναμική δομή δεδομένων για διατεταγμένη συλλογή στοιχείων.
- Όχι απαραίτητα του ίδιου τύπου.
- Τα στοιχεία μέσα σε [ ].
- Χρησιμοποιούμε λίστες στην python για την αναπαράσταση πινάκων.

**Χαρακτηριστικά:**

- Ένα Όνομα
- Πολλές Θέσεις
- Μια Δομή

```python
list = [5, 7, 8, 9, 3]

list[2] = 4
list[0] = input("Δώσε τιμή: ")
```

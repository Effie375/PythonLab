## 4.2 Η εντολή While

---

Mη-Προκαθορισμένος Αριθμός Επαναλήψεων.

```python
while (συνθήκη):
  commands
```

**Παράδειγμα:**

```python
# Αρχικοποίηση μεταβλητής
x = 0
# Για όσο το x είναι διάφορο του 3
while x != 3:
  # Εκτύπωση Hello
  print("Hello")
  # Αυξάνουμε το x κατά 3
  x += 3
```

## 4.3 Η εντολή While

---

Mη-Προκαθορισμένος Αριθμός Επαναλήψεων.

```python
while (συνθήκη):
  commands
```

**Παράδειγμα:**

```python
x = 0
while x != 3:
  print("Hello")
  x += 3
```
